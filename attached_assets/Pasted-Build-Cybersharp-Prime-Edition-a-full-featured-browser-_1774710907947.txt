## WEBGPU RENDER PIPELINE

```wgsl
// Implement these render passes in order:

Pass 1: Shadow Map Generation
  - Render scene depth from sun/moon direction
  - 2048x2048 depth texture

Pass 2: G-Buffer Pass (Deferred)
  - Output: albedo (rgba8), normal (rgba16f), depth (r32f), emissive (rgba8)
  - Vertex shader: MVP transform, UV, normal matrix
  - Fragment shader: texture sample + normal map

Pass 3: Lighting Pass (Compute)
  - Read G-buffer textures
  - Apply directional + 256 point lights
  - Shadow map lookup with PCF soft shadows
  - Output: HDR color buffer

Pass 4: SSR (Screen-Space Reflections)
  - Ray march in screen space using depth buffer
  - Blend with roughness map

Pass 5: Particle System (Compute)
  - Rain: 4000 particles, gravity + wind, collision with ground plane
  - Sparks/muzzle flash: burst emitters

Pass 6: Post-Processing (full-screen quad)
  - Bloom: threshold extract → dual kawase blur → additive blend
  - Tone mapping: ACES filmic
  - Chromatic aberration
  - Film grain
  - Vignette
  - CRT scanlines (optional)

Pass 7: UI Overlay (2D)
  - Render HUD elements using msdf-bmfont text
  - Holographic panel shader (flicker + scanline)
```
